### 英文單字/短語對照表 (含科技/資訊領域常譯)

| 英文單字 / 短語 | 詞性類別 | 中文說明與常見用法 |
| :--- | :--- | :--- |
| **Federation** | 名詞 (n.) | 聯邦、聯盟、聯合會；（資訊）聯合架構或身份聯邦 |
| **configures** | 動詞 (v.) | 配置、設定、調整（主詞為第三人稱單數） |
| **wildcard** | 名詞 (n.) / 形容詞 (adj.) | 萬用字元（如 `*` 或 `?`）、萬用牌；通配的 |
| **past** | 形容詞 / 名詞 / 介係詞 / 副詞 | 過去的；過去；經過、越過 |
| **demand** | 名詞 (n.) / 動詞 (v.) | 需求、要求 |
| **mechanisms** | 名詞 (n. 複數) | 機制、結構、原理、運作方式 |
| **moderation** | 名詞 (n.) | 適度、節制；（網路）內容審查/內容管理 |
| **fits** | 動詞 (v.) / 名詞 (n. 複數) | 適合、符合、容納（動詞第三人稱單數）；突發、發作（名詞） |
| **nested** | 形容詞 (adj.) | 巢狀的、內嵌的、嵌入的（例如：巢狀迴圈 nested loop） |
| **zillions** | 名詞 (n. 複數，非正式) | 龐大的數字、無數、千千萬萬 |
| **excluded** | 形容詞 (adj.) / 動詞過去式 | 被排除的、不包括的；排除（過去式/過去分詞） |
| **A legacy bank** | 名詞短語 (Noun phrase) | 傳統銀行、老牌銀行（通常指系統較舊、轉型中的傳統金融機構） |
| **impact** | 名詞 (n.) / 動詞 (v.) | 影響、衝擊、效果 |
| **the recommended way** | 名詞短語 (Noun phrase) | 推薦的方法、建議的作法 |
| **fundamentals** | 名詞 (n. 複數) | 基本原理、根本、基礎知識 |
| **monotonically** | 副詞 (adv.) | 單調地（數學/資訊常指「單調遞增」或「單調遞減」，數值只增不減或只減不增） |
| **regarding** | 介係詞 (prep.) | 關於、至於 |
| **eligible** | 形容詞 (adj.) | 有資格的、合格的 |
| **Instead** | 副詞 (adv.) | 反之、代替、改為（通常放在句首或句尾） |
| **rarely** | 副詞 (adv.) | 很少、罕見地、難得 |
| **the full profile documents** | 名詞短語 (Noun phrase) | 完整檔案文件、完整設定檔文件 |
| **combined** | 形容詞 (adj.) / 動詞過去式 | 組合的、結合的；結合、聯合（過去式/過去分詞） |
| **exclude** | 動詞 (v.) | 排除、不包括 |
| **via** | 介係詞 (prep.) | 經由、透過、憑藉 |
| **versus** | 介係詞 (prep.) | 對、對抗、與...相對（通常縮寫為 **vs.**） |
| **consistency** | 名詞 (n.) | 一致性、連貫性（資料庫中 ACID 的 C） |
| **isolation** | 名詞 (n.) | 隔離、孤立、獨立性（資料庫中 ACID 的 I） |
| **halts** | 動詞 (v.) / 名詞 (n. 複數) | 停止、終止、暫停（動詞第三人稱單數） |
| **occurrence** | 名詞 (n.) | 發生、事件、出現 |
| **migrating** | 動詞現在分詞 / 動名詞 | 遷移、移轉（系統或資料從舊環境搬到新環境的過程） |
| **a handful of** | 形容詞短語 | 少數的、一把的、寥寥無幾的 |
| **bestsellers** | 名詞 (n. 複數) | 暢銷書、暢銷商品 |
| **drawback** | 名詞 (n.) | 缺點、短處、不利條件 |
